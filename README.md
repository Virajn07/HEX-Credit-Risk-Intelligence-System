# HEX-Credit: AI Credit Risk Intelligence System

HEX-Credit is an AI-powered banking analytics platform that assists credit analysts and relationship managers in evaluating loan applications and monitoring portfolio risk.

## Features

- Machine Learning risk prediction
- SHAP explainability
- Counterfactual loan adjustment engine
- AI loan explanation (LLM)
- AI customer portfolio review
- Portfolio risk dashboard
- Relationship manager AI chatbot
- Loan & customer report generation

## Technologies Used

Python, Flask, Scikit-Learn, SHAP, Pandas, OpenRouter LLM, Chart.js

## System Architecture

Loan Data → ML Model → Risk Prediction → Explainable AI → Counterfactual Engine → LLM Analysis → Dashboard

## Use Case

The system simulates credit risk analytics platforms used by financial institutions to support lending decisions and portfolio monitoring.

## Setup

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open browser:

http://127.0.0.1:5000
